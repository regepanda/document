## 变量与基本类型
### 引用
```
1)、引用必须被初始化,且定义引用时，引用与初始值互相绑定，而不是将初始值拷贝给引用
2)、引用不是对象
3)、引用只能绑定在对象上，而不能与字面值或某个表达式的计算结果绑定在一起，而且要与绑定的对象类型严格匹配
4)、引用只能绑定到一个对象，一旦初始化，就不能再重新绑定到另外一个对象
```
### 指针
```
1)、如果指针要访问某个对象，需要加上解引用符*
2)、可用nullptr初始化指针，与【int *pi = 0】一样
3)、void*：一种特殊指针类型，可以存放任意对象的地址
4)、指针本身是一个常量并不意味着不能通过指针修改其所指对象的值，能否这样做完全取决于依赖于所指对象的类型【conast】
```
### const限定符
```
1)、默认状态下const对象仅在文件内有效,如果在一个文件中定义了一个const对象要想在其他文件中使用，重复定义，可以如下：
    extern const int bufSize = fcn(); file_1.cc【定义并初始化一个常量，并且此常量能在其他文件访问】
    extern const int bufSize; file_1.h【与上一个文件中的bufZize是同一个】
2)、 顶层const表示指针本身是一个常量，底层const表示指针所指向的对象是一个常量
```
