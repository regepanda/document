###生成器
```
1、比如在迭代的时候生成一个数组，你可能会给这个数组开辟一个数组长度的空间，但是用生成器会即时的为数组的每一个元素开辟一个空间，节省空间，性能提升很多
   eg:function makeRange($length)
      {
        for($i=0;$i<$length;$i++)
        {
            yield $i;
        }
      }
      foreach(makeRange(10000) as $i)
      {
        echo $i;
      }
```
